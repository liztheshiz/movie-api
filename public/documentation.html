Welcome to Cinema Database!

This API essentially provides an interactive database of information on films, directors, and genres.
Additionally, users can register to create and manage a list of their favorite films.


Table URL endpoints and their req/res types:

Business Logic URL HTTP Method Request body data format Response body data format
Get list of all movies /movies GET none JSON object with list of movies with their info
Get data about a single movie by title /movies/[Title] GET none JSON object with movie info (description, genre, director, image URL, whether it's featured or not) 
                            
                                {
                                    title: "",
                                    description: "",
                                    genre: "",
                                    director: "",
                                    imageUrl: "",
                                    featured: ""
                                }
Get data about a genre (description) by name/title /movies/genres/[Genre] GET none Text message with genre description
Get data about a director by name /movies/directors/[Director] GET none JSON object with director info (bio, birth year, death year) 
                            
                                {
                                    name: "",
                                    bio: "",
                                    birthYear: "",
                                    deathYear: ""
                                }
Add new user /users POST JSON object with username and password 
                            
                                {
                                    name: "",
                                    password: ""
                                }
JSON object with username, password, and favorite movies list (begins empty) 
                            
                                {
                                    name: "",
                                    password: "",
                                    topMovies: {}
                                }
Update user's username /users/[Current username]/[New username] PUT none Text message indicating user's old and new username
Add movie to user's favorites list /users/[Username]/topMovies POST JSON object with data about movie (description, genre, director, image URL, whether it's featured or not) 
                            
                                {
                                    title: "",
                                    description: "",
                                    genre: "",
                                    director: "",
                                    imageUrl: "",
                                    featured: ""
                                }
JSON object with data about movie that was added (description, genre, director, image URL, whether it's featured or not) 
                            
                                {
                                    title: "",
                                    description: "",
                                    genre: "",
                                    director: "",
                                    imageUrl: "",
                                    featured: ""
                                }
Remove movie from user's favorites list /users/[Username]/topMovies/[Title] DELETE none Text message indicating movie was removed
Remove user /users/[Username] DELETE none Text message indicating user was removed
